# myapp
This is a node js app.
```
Instructions:

(1) Make sure you have installed nodejs and available in your path
(2) Clone depository to your local node app directory (example: /app)
(3) cd /app; npm install 
(4) "node app.js" will run application with logging ON *** application will be listening on port 3000 ***
(5) "NODE_ENV=production node app.js" to run application without logging
(6) "node test/test.js" to test the application ***NOTE: Ctrl-C to break out of test, unless test didn't pass with assertions***


```
